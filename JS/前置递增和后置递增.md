## 前置递增和后置递增的区别
---
1. 前置递增

    >在计算机科学领域，有一种情况被称为副效应：变量里面保存的值在语句执行之前发生了改变。
     前置递增就是这种情况，即先递增再赋值。 看代码 

    ```javascript
    var num = 19;
    var anoNum = ++num + 1;
    console.log(anoNum); //21
    ```

2. 后置递增呢，则刚好相反，赋值之后再进行自增

    ```javascript
    var num = 19;
    var anoNum = num++ + 1;
    console.log(anoNum); //20
    ```

**其实，前置自增和赋值操作符的运算优先级是相等的，所以代码运行时，会先进行赋值操作，然后在进行递增**
